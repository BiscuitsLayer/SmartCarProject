# Smart Car Project

OpenGL based self-driving car (with self made neural network)

Make sure you installed:
1. Git
2. Powershell
3. GNU Make for Windows 
4. OpenGL

To execute:

    git clone https://github.com/BiscuitsLayer/SmartCarProject.git
    cd ./SmartCarProject
    git submodule update --init --recursive
    ./run.ps1
